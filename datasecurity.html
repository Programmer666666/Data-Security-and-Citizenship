What is data security?

Data security is where you protect digital data for examples those in data bases. They are protected from unauthorized users, unwanted actions or theft.




Why is it important?

Data security is very important because it can prevent unwanted actions and even theft of your personal information. Data security is a topic that is very important and needs to be talked about a lot. Data security can even be physical too with protecting hardware and storage devices.


The types of data security.

Some of the most common types of data security are data encryption, access control, authentication etc. authentication is where you identify users before they have access to the data. The data can consist of passwords, pin numbers, security tokens, biometrics etc. encryption is when a computer turns text into unreadable format with encryption keys. And only someone with the proper corresponding keys can access this information. Access control is a type of data where it limits both physical and digital access to some critical system and data.


Data security risks

There can be many risks with data security that are easy to miss or accidentally do. some of these can include accidentally sharing data you didn’t mean to, bad password hygiene, phishing emails etc.


How you can resolve these issues

There are ways to prevent these data risks from happening for example with accidentally sharing data what you need to do is make sure if you are sending sensitive info that you’re sending it to the right person and that the person is trusted and can see the info. Never show any sensitive info to someone even if they are trusted. Bad password hygiene is an important and big on you should always have strong passwords that are not easy to crack and never have the same one for a different account. For the last one phishing email this is one that you can get caught into easily without even knowing. To prevent this, you need to not click on suspicious links even if its sent by a friend message the friend to make sure it was them and not a fake.
